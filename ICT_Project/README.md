# Cafe Website Project

This is my first semester ICT project. It is a cafe-based website created as a group project to practice web development skills using HTML, CSS, and JavaScript.

## About The Project

The website is designed for a cafe and includes multiple pages such as the home page, menu page, bakery page, coffee page, blended drinks page, contact page, login page, signup page, and cart page.

This project helped me learn how to:
- Structure a multi-page website
- Design pages using CSS
- Use JavaScript for interactive behavior
- Organize files for a front-end web project
- Work on a project with teammates

## Features

- Home page for the cafe
- Menu browsing
- Coffee and blended drinks pages
- Bakery items page
- Contact page
- Login and signup forms
- Cart page
- Product images and styling

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

- `index.html` - Home page
- `menu.html` - Main menu page
- `bakery.html` - Bakery items page
- `Coffees.html` - Coffee menu page
- `blendedDrinks.html` - Blended drinks page
- `contact.html` - Contact page
- `login.html` - Login page
- `signup.html` - Signup page
- `cart.html` - Cart page
- `java.js` - JavaScript functionality
- `*.css` - Styling files
- `*.avif` - Product images

## How To Run The Project

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your browser
